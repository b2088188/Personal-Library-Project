## General info
Purpose: 專案的目的是給予使用者更易於操作的書籍瀏覽 UI, 並提供真實購買連結供使用者進行連結購買。
Motivation: 開發這個專案主要的動機是將所學的基礎 HTML / CSS / JavaScript, 進行實際的開發和應用, 有良好的基礎才能在學無止境的路途上走的順遂一些。
